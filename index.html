SMART SCHEDULLING
Google Form → Google Sheets Trigger → Google Calendar → IF → Crear evento/Ocupado

Sistema de agendamiento automático que permite a los clientes reservar reuniones directamente desde un formulario, verificando disponibilidad en tiempo real para evitar choques de horario. Si el horario está libre, crea el evento automáticamente en el calendario y envía confirmación por correo; si está ocupado, avisa al cliente para que elija otro horario. Ideal para profesionales independientes, clínicas, talleres o cualquier negocio que atiende con horas agendadas. Elimina la necesidad de coordinar manualmente por WhatsApp o llamadas.

WHATSAPP BOT
Twilio Webhook → n8n → Respond 200 → Switch (keyword routing) → Respuesta automática

Bot de WhatsApp que atiende clientes automáticamente las 24 horas, sin intervención humana. Detecta la intención del mensaje y responde en consecuencia: si el cliente quiere hacer un pedido, notifica al dueño del negocio al instante con el detalle completo; si quiere agendar, le envía el link del formulario directo; y para cualquier otro mensaje, responde con las opciones disponibles. Las respuestas, flujos y acciones son completamente modificables según las necesidades de cada negocio. Ideal para restaurantes, tiendas, clínicas o cualquier negocio que recibe consultas por WhatsApp fuera del horario laboral. Elimina la necesidad de estar pendiente del celular para no perder clientes.

CONTENT INTELLIGENCE
RSS (HN + Reddit) → Dedup Sheets → Filtro IA (Groq) → Resumen ES → Telegram

Sistema de monitoreo y resumen de contenido tech con IA, diseñado para dueños de negocios que quieren mantenerse al día sin perder tiempo filtrando información irrelevante. Cada 6 horas monitorea múltiples fuentes RSS simultáneamente, descarta artículos ya procesados, usa IA para filtrar solo lo relevante para el negocio, genera resúmenes en español con el valor concreto para el dueño, y entrega el resultado directo a Telegram, sin intervención humana. Este sistema puede adaptarse para monitorear noticias de un sector específico, seguir tendencias de competidores, rastrear cambios regulatorios o legales relevantes para el negocio, o mantenerse al tanto de lanzamientos de herramientas y tecnologías del rubro.
